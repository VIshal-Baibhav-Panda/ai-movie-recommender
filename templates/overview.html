About This Project

This AI Movie Recommender uses advanced AI models to suggest movies based on your input. It analyzes movie context and generates intelligent recommendations.

It also integrates TMDB API to fetch real movie posters, making the experience more interactive and visually appealing.

Built using Flask, OpenRouter API, and modern UI design principles.
